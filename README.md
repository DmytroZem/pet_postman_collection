# Postman Collection JSON Workflow

This project manages the export, versioning, and collaboration on **Postman Collection JSON** files. Follow the standardized workflow to keep collections organized, reviewed, and up-to-date.

## 🚀 Quick Start

1. Clone the repo
2. Create your feature branch
3. Export your Postman collection as JSON
4. Commit and push following the workflow below
